Online Compiler

The moto behind this project was to design a web based application
that can be used as an online compiler for languages like C/C++
Java and Python by any individual or in any programing contest.
The main idea was to link rich command line feature of a linux
with a web page using PHP.to design front-end HTMl and CSS along-with
JavaScript is used.To handle back-end database MYSQL is used.
Features like user registration and auto detect language is added to it.
